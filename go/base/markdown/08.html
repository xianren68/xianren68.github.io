08_包 ​

go中的代码以package的形式组织，类似于java的package,一个包通常以一个文件夹的形式组织，内部的所有.go文件都是这个包中的文件 每个.go都需要在第一行表明所属的包名。

如上所有文件的第一行都需要package repository

INFO

go一个包中文件可以互相访问，而不要导入之类的操作，所以它们之间变量的定义也不能重复
